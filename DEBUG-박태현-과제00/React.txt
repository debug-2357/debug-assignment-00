React

1. React
React : JAVASCRIPT 라이브러리로써, 싱글페이지 애플리케이션의 UI를 생성하는데 집중한 라이브러리
프레임워크와 라이브러리의 차이 : 제어 방식의 차이가 있다
프레임워크의 경우 : 제어 흐름 -> 프레임워크
라이브러리의 경우 : 제어 흐름 -> 개발자
React 특징 1. JSX(JavaScript XML) -> 
	      2. VIRTUAL DOM 
	      3. 단방향 데이터 바인딩 : 양방향 데이터 바인징보다 효율성이 높다
	      4. 컴포넌트 기반 : 코드의 재사용성을 높였다
DOM(Document Object Model)
